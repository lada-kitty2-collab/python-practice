# python-practice
Учебные проекты на Python. Каждый файл показывает практику с циклами, условиями. списками и простыми алгоритмами.
My Python practice projects
## Файлы
### 1. atm_simulator.py
- Симулятор банкомата.
- Использует 'while', 'if/else', 'input' для проверки PIN-кода, баланса и снятия денег.
### 2. turnstile.py
- Турникет с картой.
- Пример работы с 'while', 'True/False' переменными
### 3. card_payment.py
- Симулятор оплаты с использованием карты
- Пользователь прикладывает карту
- Проверяется наличие средств для оплаты
- Используется циклы 'while', условия 'if/else' и переменные 'True/False'
- Пример работы с пользовательсиким вводом ('input') и логикой принятия решений.
- ### light_switch_simulator.py
- Симулятор управления освещением с тремя переключателями
- Есть:
  - left_switch — аварийный свет
  - left_switch_sec — основной свет
  - right_switch — выключение всех светов
- Использует:
  - Цикл while для постоянного ожидания действий пользователя
  - Условия if/elif для проверки состояния переключателей
  - Переменную state для отслеживания текущего состояния
  - Ввод пользователя (`input`) для управления светом
  - Задержки времени через time.sleep для имитации реакции света
- Демонстрирует навыки:
  - Работа с переменными
  - Управление состояниями
  - Взаимодействие с пользователем через консоль
